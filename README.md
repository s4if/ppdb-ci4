# ppdb-ci4
Upgrade program PPDB SMA/SMK Ihsanul Fikri dengan framework CI4

### Target
- Fitur yang sama dengan versi ppdb-smk
- Dependency Terbaru
- Kompatibel PHP 8+

### Dependency
- Doctrine ORM
- gregwar/captcha (or alternative)
- mikehaertl/phpwkhtmltopdf
- phpoffice/phpspreadsheet
- Twig (New?)
